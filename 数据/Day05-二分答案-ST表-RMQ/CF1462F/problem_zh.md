# CF1462F · The Treasure of The Segments

- 难度：1800
- 标签：binary search、data structures、greedy

- 时间限制：3 seconds　内存限制：256 megabytes

## 中文题意

给定 $n$ 条线段，第 $i$ 条用两个整数 $l_i$、$r_i$ 描述（左右端点坐标）。

称一个含 $k$ 条线段的集合是「好的」，如果集合中存在某一条线段 $[l_i, r_i]$ 与集合中每一条线段都相交（相交指有公共点或公共线段）。例如 $\{[1,4],[2,3],[3,6]\}$ 是好的，因为 $[2,3]$ 与其余每条都相交；而 $\{[1,2],[2,3],[3,5],[4,5]\}$ 不是好的。

问：最少删除多少条线段，才能使剩下的线段构成一个好的集合？

本题包含 $t$ 组测试数据。

## 输入格式（中文）

第一行一个整数 $t$（$1 \le t \le 2\cdot10^5$）——测试数据组数。

每组数据第一行一个整数 $n$（$1 \le n \le 2\cdot10^5$）——线段数；随后 $n$ 行，每行两个整数 $l$、$r$（$1 \le l \le r \le 10^9$）描述一条线段。

保证所有测试数据的 $n$ 之和不超过 $2\cdot10^5$。

## 输出格式（中文）

对每组数据输出一个整数——为使剩余线段构成好的集合，最少需要删除的线段数。
