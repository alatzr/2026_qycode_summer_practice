# CF977E Cyclic Components 题解

## 题目大意

统计无向图中有多少个连通块本身恰好是一个简单环。连通块里不能有额外的分支或弦。

## 思路

一个连通无向图恰好是一个环，当且仅当这个连通块中每个点的度数都是 `2`。

读入每条边时：

1. 用并查集合并边的两个端点；
2. 累加两个端点的度数。

读入完成后遍历所有点。如果某个点的度数不是 `2`，就把它所在的连通块标记为不合法。最后只统计仍然合法的并查集根。

注意：“连通块边数等于点数”只能说它是单环图，不代表整个连通块是环。三角形后面接一条链就是反例。

## 复杂度

时间复杂度为 `O((n+m) alpha(n))`，空间复杂度为 `O(n)`。
