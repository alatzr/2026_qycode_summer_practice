# CF722C Destroying Array

- 难度：1600
- 标签：DSU、离线、逆序处理
- 链接：https://codeforces.com/problemset/problem/722/C
- 出现位置：Day12-Kruskal-离线排序-路径贡献

## 中文题意

给定一个含 `n` 个非负整数的数组，按给定排列的顺序依次删除位置。每次删除后，求不经过已删除位置的连续段中，最大的元素和。空段的和为 `0`。

## 输入格式

第一行 `n` (`1 <= n <= 100000`)。第二行为 `a_i` (`0 <= a_i <= 10^9`)。第三行是 `1..n` 的一个排列，表示删除顺序。

## 输出格式

输出 `n` 行，第 `i` 行是前 `i` 次删除后的最大连续段和。

## 样例

```text
4
1 3 2 5
3 4 1 2
```

```text
5
4
3
0
```
