# CF1213G Path Queries

- 难度：1800
- 标签：离线排序、DSU、路径贡献
- 链接：https://codeforces.com/problemset/problem/1213/G

## 中文题意

给定一棵 `n` 个顶点的带权树和 `m` 个询问。对每个阈值 `q_i`，求有多少对 `u<v` 满足 `u` 到 `v` 的路径上最大边权不超过 `q_i`。

## 输入格式

`1 <= n,m <= 200000`。接下来 `n-1` 行给出树边，边权为 `1..200000`。最后一行给出 `m` 个 `1..200000` 之间的询问。

## 输出格式

按输入顺序输出每个询问的点对数。

## 样例

```text
3 3
1 2 1
2 3 2
1 3 2
```

```text
1 3 3
```
