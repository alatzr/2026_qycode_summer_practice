# CF1081D Maximum Distance

- 难度：1800
- 标签：Kruskal、DSU、瓶颈路径
- 链接：https://codeforces.com/problemset/problem/1081/D

## 中文题意

给定一张带权连通无向图和 `k` 个特殊顶点。一条路径的代价是路径上的最大边权；两点距离是所有路径代价的最小值。

对每个特殊顶点，求它到最远特殊顶点的距离。

## 输入格式

`2 <= k <= n <= 100000`，`n-1 <= m <= 100000`，边权 `1..10^9`。图中允许自环和平行边，保证连通。

## 输出格式

输出 `k` 个整数，分别是各特殊顶点的答案。

## 样例

```text
4 5 3
1 2 3
1 2 5
4 2 1
2 3 2
1 4 4
1 3 3
```

```text
3 3 3
```
