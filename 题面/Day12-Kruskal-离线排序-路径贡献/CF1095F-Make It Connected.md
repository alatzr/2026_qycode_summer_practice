# CF1095F Make It Connected

- 难度：1900
- 标签：Kruskal、DSU、候选边缩减、贪心
- 链接：https://codeforces.com/problemset/problem/1095/F
- 出现位置：Day12-Kruskal-离线排序-路径贡献

## 中文题意

顶点 `i` 有数值 `a_i`。任意两个不同顶点 `x,y` 都可以花 `a_x+a_y` 连边；另外有 `m` 个可选的特价方案，可以花 `w` 连接指定的 `x,y`。求使整张图连通的最小总代价。

## 输入格式

第一行 `n,m` (`1 <= n <= 200000`, `0 <= m <= 200000`)。第二行 `a_i` (`1 <= a_i <= 10^12`)。接下来 `m` 行为 `x,y,w` (`x != y`, `1 <= w <= 10^12`)。

## 输出格式

输出最小总代价。

## 样例

```text
3 2
1 3 5
1 2 2
2 3 1
```

```text
3
```
